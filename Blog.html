Come creare i template per i post

Di Mario Rossi - Pubblicato il 25-10-2020

La prima cosa da fare è suddividire in blocchi logici la pagina

Le varie parti saranno divise in:

  • Contenuti di instestazioni nel tag <Header>
  • Contenuti centrali presenti nel tag <Main>
  • Contenuti finali presenti nel tag <Footer>

Formatazzione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare nel CSS. I passi saranno:

  1. Progettare il markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere l'html dei tag esterni ad ogni specifico testo il tag appropriato
  3. Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale
